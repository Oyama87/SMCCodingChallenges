    Front End Resources

  • MDN: Getting started with HTML
    • Entry point for the Mozilla Developer Network HTML tutorials.
    Start here if you have no experience with web development.
  • MDN: How CSS Works.
    • Introduction to how CSS is applied to HTML elements. A little
    heavy on theory, but highly recommended to start here after you
    have gotten a little bit of experience with HTML.
  • MDN: CSS Syntax.
    • Beginning tutorial for writing your own CSS rules.
  • Complete Guide To FlexBox
    • Simple but thorough guide to using FlexBox
    properties and values.
  • Flexbox Froggy
    • A fun game that teaches you to
    use Flexbox to position elements.
  • W3Schools: How-To Guides
    • Large collection of guides on modern front
    end web techniques such as drop-down menus,
    navigation bars, parallax scrolling, and more.
  • React Express
    • A guide to getting started with React.
    Prior experience with HTML, CSS, and
    JavaScript highly recommended.

    Logic Resources

  • You Don't Know JavaScript
    • GitHub Repository containing a book on advanced JavaScript
    programming concepts and techniques.
  • C/C++ I/O Libraries
    • Quick Reference for C/C++ Input/Output stream member
    values and functions.

    Mobile Resources

  • Official React-Native Documentation
    • React-Native is a technology invented by Facebook
    that allows developers to write native mobile apps
    for iOS and Android using only JavaScript and JSX.
  • Expo
    • Expo is an Integrated Development Environment
    specifically designed for mobile app development
    using React-Native.
